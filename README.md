### light 
运行命令 light 能看见帮助信息 help

### 安装
```
  npm install -g @daes/rollup-cli
```

安装后，您可以通过简单地运行来验证它是否已正确安装light，这应该会向您显示一条列出所有可用命令的帮助消息。
您可以使用以下命令检查您是否拥有正确的版本：`light --version`

### Creating a Project
To create a new project, run:
```
  light create hello-world
```
